# ☕ Кофета — меню кофейни

Красивое и удобное веб-меню для кофейни **«Кофета»**.  
Проект создан для того, чтобы гости могли быстро посмотреть напитки, десерты в приятном современном интерфейсе.

---

## ✨ О проекте

**Кофета** — это цифровое меню кофейни с акцентом на простоту, атмосферу и удобство.

В меню можно представить:

- кофе и авторские напитки;
- чай и сезонные напитки;
- десерты;
